## 3.2 计算集群分类

|  集群名  | 主要用途 |  说明    |
| --------- | -----|  ------- |
| 单节点集群<br/>Single-node Cluster | 测试、开发 | 单个节点同时充当管理节点、计算节点|
| 内联集群<br/>Inline Cluster |  | 集群包括1个或多个头节点、若干个计算节点。也分为静态分配、动态调度两类 |
| 动态集群<br/>Dynamic Cluster |  | 管理节点包括1个或多个头节点，计算节点则由外部调度器分配资源 |
