- 天然分布式设计

程序表达能力：
- 结构化程序：顺序、分支、循环
- goto语句

scalebox程序，通过消息发送，可实现类似于goto语句的功能。从而支持完整的应用逻辑表达。

模块间的应用逻辑表达：基于消息发送，实现控制流，类似于goto的功能
